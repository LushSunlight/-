新酷产品第一时间免费试玩，还有众多优质达人分享独到生活经验，快来新浪众测，体验各领域最前沿、最有趣、最好玩的产品吧~！下载客户端还能获得专享福利哦！据 Neowin 报道，在 2021 年审查 Windows 11 时，该网站给它打了 6.510 的分数，同时表示“简化 UI 并不是糟糕的主意，但半生不熟的方式并没有带来真正意义上的用户体验”。遗憾的是，六个月后它仍然没有改变。正如我们在最近的 Insider 版本中看到的那样，微软似乎有意在添加新功能，但同时忽略了现有版本中的 UI 不一致和缺乏基本功能的情况。截图显示中的所有应用程序都是微软开发的内置应用，但我们看到 UI 和 UX 始终不一致，就好像没有高级领导来监督操作系统的设计一样。甚至图标的样式也不同，有些边框较粗，有些则更简约。就好像每个开发团队都在自己的孤岛中工作，只专注于他们认为看起来漂亮的东西，而不去考虑 Windows 11 的整体美感如何。事实上，这个问题不仅限于图标。我们发现，Windows 10 的旧菜单遍布 Windows 11 整个操作系统，菜单选项基本上已经从以前的 Windows 版本中提升和转移，但根本没有考虑整体 UX。所有这些都表明 Windows 11 明显缺乏设计标准化，这对于被数百万人使用的操作系统来说并不是好的感官体验。Windows 11 还存在缺少基本功能的问题。微软仍然以荒谬的理由拒绝让人们移动 Windows 11 任务栏，但却开发了记事本可转动的设置按钮齿轮。Windows 11 开发团队很庞大，由多个子团队组成，不同的人从事不同的工作。但这种类型的优先排序会导质量参差不齐。那么什么是优先的基础性功能？微软反馈中心那里有大量的项目有成千上万的赞成票，理想情况下，微软应该在开发方面以更高的优先级跟踪这些项目。然而，这些请求中的许多已经被搁置了几个月，都没有来自微软的更新，说明微软有意忽视了这些反馈。Windows 11 显然缺乏集中监督，应该负责管理优先级、指导缺失功能的开发和标准化 UI 设计。现有方式的问题在于，每个团队将继续以孤立的方式在自己的重点领域进行开发，虽然这可能会导致添加大量功能，但它可能不是人们实际需要的东西。因为 UX 的不一致，Windows 11 不同组件的集合，可能导致该操作系统很难具有凝聚力。微软是时候考虑做出改变了。
