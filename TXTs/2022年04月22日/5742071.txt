原标题：苹果宣布停止macOS Server服务财经网科技4月22日讯，据IT之家消息，苹果今天在一份支持文档中宣布，macOS Server 将于 2022 年 4 月 21 日停止。苹果多年来一直在逐步淘汰 macOS Server，现在终于准备好永久关闭它了。macOS Server 可让用户将通过批量购买计划 （VPP） 获得的各种 App 分配至每台设备，无需使用用户的 Apple ID。这样即可在 iOS 设备和 Mac 电脑上安装通过 VPP 获取的各种 App，不用再配置 Apple ID 或发送邀请。用户还可以将设备上已安装的 App 迁移至用户的 Apple ID 中，而无需删除该 App 或用户数据。macOS Server 5.12.2 将是该应用的最后一个版本，macOS Server 服务现已迁移到 macOS。自 macOS High Sierra 以来，包括缓存服务器、文件共享服务器在内的流行 macOS 服务器功能已捆绑在 macOS 中，因此不再需要独立的应用。现有 macOS Server 用户可以继续下载该应用，并将其与 macOS Monterey 一起使用。责任编辑：王蒙
